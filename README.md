# MyApp

An iOS app that plays a video then automatically closes.

## Setup

1. Add your `video.mp4` file to the `MyApp/` folder
2. Push to GitHub
3. Go to Actions tab → Build IPA → Run workflow
4. Download the IPA from the artifacts

## Install IPA on iPhone (Windows)

- **AltStore**: https://altstore.io/
- **Sideloadly**: https://sideloadly.io/
- **3uTools**: http://www.3u.com/

## Notes

- Requires iOS 14.0+
- App closes automatically after video finishes
- No Apple Developer account needed for sideloading
